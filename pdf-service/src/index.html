PDF Service Documentation

A high-performance, containerized service for generating PDFs from HTML content with support for templates, custom styling, and various output options. Built with Express.js and Puppeteer.

Installation

The service is available as a Docker image for easy deployment:

docker pull blitzjb/pdf-service:latest

Quick Start

Run the PDF service using Docker:

docker run -p 3002:3000 blitzjb/pdf-service:latest

Or using Docker Compose:

services:
  pdf-service:
    image: blitzjb/pdf-service:latest
    ports:
      - "3002:3000"
    environment:
      - PORT=3000

Default Configuration

Parameter Default Value Description
format A4 Page format
landscape false Page orientation
printBackground true Include background graphics
margin 0px Page margins (all sides)
viewport 1200x800 Default viewport size

API Reference

Generate PDF

Convert HTML content to PDF with customizable options.

POST /generate-pdf

Request Body

{
  "html": "HTML content or template string",
  "options": {
    "format": "A4",
    "margin": "1cm",
    "landscape": false,
    "scale": 1.0,
    "printBackground": true
  }
}

Response

  • Content-Type: application/pdf
  • Body: PDF file content

Error Responses

{
  "error": "HTML content is required"
}
400 Bad Request 
{
  "error": "Failed to generate PDF"
}
500 Internal Server Error

Usage Examples

curl -X POST \
  http://localhost:3002/generate-pdf \
  -H 'Content-Type: application/json' \
  -d '{
    "html": "
Hello World
This is a test PDF.
",
    "options": {
      "format": "A4",
      "margin": "1cm"
    }
  }' \
  --output output.pdf
import requests

def generate_pdf(content, options=None):
    response = requests.post(
        "http://localhost:3002/generate-pdf",
        json={
            "html": content,
            "options": options or {}
        }
    )
    response.raise_for_status()
    return response.content

# Example usage
content = """

Hello World


This is a test PDF.

"""

options = {
    "format": "A4",
    "margin": "1cm"
}

pdf_content = generate_pdf(content, options)
with open("output.pdf", "wb") as f:
    f.write(pdf_content)
const fetch = require('node-fetch');
const fs = require('fs').promises;

async function generatePDF(content, options = {}) {
  const response = await fetch('http://localhost:3002/generate-pdf', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      html: content,
      options,
    }),
  });

  if (!response.ok) {
    throw new Error(`PDF generation failed: ${response.statusText}`);
  }

  return response.buffer();
}

// Example usage
async function main() {
  const content = `
    
Hello World

    
This is a test PDF.

  `;

  const options = {
    format: 'A4',
    margin: '1cm',
  };

  try {
    const pdfContent = await generatePDF(content, options);
    await fs.writeFile('output.pdf', pdfContent);
    console.log('PDF generated successfully!');
  } catch (error) {
    console.error('Error:', error.message);
  }
}

main();
